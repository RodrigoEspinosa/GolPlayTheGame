GolPlayTheGame
==============

The multiplayer game for GolPlay. Made in HaXe with HaXeNME


**Dependencias**


* Haxe + NME
* HaxePunk
* TJSON (Para utilizar con MongoDB)

**Clases fundamentales de AS3**

* flash.net
    * flash.net.NetConnection
    * flash.net.NetStream
    * flash.net.SharedObject
